# Tours-Booking App

## Live Preview Link

[https://tours-booking-rental-app.vercel.app/]

### Install packages

```bash
npm install
#or
yarn install
```

### Setup .env file

```js
DATABASE_URL=
NEXTAUTH_SECRET=

```

### Setup Prisma

````bash
npx prisma db push


### Start the app

```shell
npm run dev
#or
yarn run dev
````
